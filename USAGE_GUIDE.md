# 网络调试助手使用指南

## 一、应用简介

网络调试助手是一款用于 Android 手机端的网络调试工具，可用于 TCP / UDP 通信测试、设备联调、端口连通性验证和日志查看。

## 二、基础使用流程

### 1. 配置连接信息

1. 打开应用。
2. 选择需要的网络模式。
3. 输入目标 IP 地址和端口。

### 2. 建立连接

1. 点击连接按钮。
2. 等待状态变为已连接。
3. 若连接失败，检查 IP、端口和网络环境。

### 3. 发送测试数据

1. 在发送区域输入测试内容。
2. 点击发送按钮。
3. 在日志区查看发送结果。

### 4. 查看接收日志

1. 观察日志区域中的接收内容。
2. 根据需要切换显示格式。
3. 必要时清空日志后重新测试。

## 三、主要功能

### 1. 网络模式

- TCP Client
- TCP Server
- UDP

### 2. 日志能力

- 实时显示发送记录
- 实时显示接收记录
- 支持基础调试查看

### 3. 其他能力

- 应用设置页
- GitHub 信息展示

## 四、常见问题

### 1. 无法连接服务器

可按以下顺序排查：

- 检查手机和目标设备是否在同一网络
- 检查 IP 地址是否填写正确
- 检查端口是否开放
- 检查服务端是否已启动

### 2. 收不到数据

建议检查：

- 服务端是否有返回内容
- 当前网络模式是否选择正确
- 数据发送格式是否符合当前测试要求

### 3. 日志显示异常

建议尝试：

- 清空日志后重新连接
- 重新发送测试数据
- 切换网络模式重新验证

## 五、适用建议

- 建议优先在局域网环境中测试
- 建议先验证连通性，再进行正式联调
- 对外展示时建议使用 release APK
