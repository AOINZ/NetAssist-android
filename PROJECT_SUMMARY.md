# 项目总结

## 项目概述

- 项目名称：网络调试助手
- 平台：Android
- 当前版本：v1.1.0
- 项目类型：手机端网络调试工具

## 项目目标

本项目用于构建一款适合 Android 手机端使用的网络调试工具，重点支持基础 TCP / UDP 通信测试、连接验证、日志查看和移动场景下的快速联调。

## 已完成内容

- 安卓项目基础结构搭建
- TCP Client / TCP Server / UDP 模式支持
- 基础连接与发送接收流程
- 应用设置页与 GitHub 展示入口
- Debug / Release APK 构建
- 发布签名配置
