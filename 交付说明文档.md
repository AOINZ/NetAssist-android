## 3. 应用功能简介

网络调试助手用于安卓手机端网络通信联调，支持常见 TCP / UDP 调试场景，适合：

- 服务器连通性验证
- TCP 客户端通信测试
- UDP 报文收发测试
- 调试日志查看

## 4. 安装方式

1. 将 APK 文件发送到安卓手机。
2. 在手机上打开 APK 文件。
3. 如系统提示“禁止安装未知来源应用”，根据系统引导允许本次安装。
4. 安装完成后，打开“网络调试助手”即可使用。

## 5. 使用说明

1. 进入应用后，在主界面填写目标 IP 和端口。
2. 选择需要的协议模式。
3. 点击“连接”建立通信。
4. 在发送区输入测试内容并发送。
5. 在日志区查看接收与发送记录。
6. 在“设置”页可查看 GitHub 地址。
