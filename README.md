# 网络调试助手

一款用于 Android 手机端的网络调试工具，适合 TCP / UDP 联调、连通性验证和日志查看。

## 功能特性

- 支持 TCP Client、TCP Server、UDP
- 支持连接测试与基础数据收发
- 支持日志查看
- 支持手机端现场调试

## 快速开始

### 直接安装 APK

- [点击下载 Release APK](releases/%E7%BD%91%E7%BB%9C%E8%B0%83%E8%AF%95%E5%8A%A9%E6%89%8B_v1.1.0_release.apk)

### 本地构建

1. 使用 Android Studio 打开项目
2. 等待 Gradle 同步完成
3. 运行项目或构建 APK

## 开发环境

- Android SDK 33
- Java 8+
- Gradle
